# 🎓 Student Grade Analyzer (NumPy)

## 📌 Project Description
This project analyzes student scores using NumPy and provides useful statistics such as mean, median, and standard deviation. It also assigns grades automatically.

## 🚀 Features
- Calculate mean, median, and standard deviation
- Identify highest and lowest scores
- Automatically assign grades (A–F)
- Display results clearly

## 🛠️ Technologies Used
- Python
- NumPy

## ▶️ How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the script:
   python main.py

## 📊 Example Output
- Mean Score
- Median Score
- Student Grades
